# \### 📄 输出文件说明

# 

# \- \*\*输出位置\*\*：SRT文件保存在与输入文件相同的位置

# \- \*\*文件命名\*\*：与原视频文件同名，扩展名为.srt

# \- \*\*示例\*\*：

# &nbsp; - 输入：`D:\\Videos\\movie.mp4`

# &nbsp; - 输出：`D:\\Videos\\movie.srt`

# \## 🌍 多语言翻译功能

# 

# \### 支持的功能：

# \- ✅ \*\*转录模式\*\* - 保持原语言

# \- ✅ \*\*翻译模式\*\* - 将任何语言翻译成英文

# \- ⚠️ \*\*翻译限制\*\* - 只能翻译成英文，不能翻译成其他语言

# \- ✅ \*\*源语言选择\*\* - 指定输入语言（可选）

# \- ✅ \*\*目标语言选择\*\* - 显示但不影响翻译结果（固定为英文）

# 

# \### 翻译功能说明：

# \- \*\*转录模式\*\*：识别语音并保持原语言输出

# \- \*\*翻译模式\*\*：识别语音并翻译成英文输出

# \- \*\*重要提示\*\*：faster\_whisper 的翻译功能只能输出英文，这是模型本身的限制

# 

# \### 使用示例：

# 1\. \*\*中文转录\*\* - 源语言：中文，任务：转录 → 输出中文

# 2\. \*\*中文翻译\*\* - 源语言：中文，任务：翻译 → 输出英文

# 3\. \*\*日语翻译\*\* - 源语言：日语，任务：翻译 → 输出英文

# 4\. \*\*自动检测\*\* - 源语言：自动检测，任务：转录 → 保持原语言

# 

# \### ⚠️ 重要提示：

# \- \*\*所有输出均为英文格式\*\*：无论选择哪种模式，最终输出的SRT文件都使用英文格式

# \- \*\*时间轴格式\*\*：使用标准的SRT时间轴格式

# \- \*\*字幕编号\*\*：自动编号字幕段落
